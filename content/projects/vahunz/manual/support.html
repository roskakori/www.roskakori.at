Updates and Support

New versions will be uploaded to Aminet and can be retrieved from aminet:dev/misc/vahunz.lha, or from the support page at http://www.giga.or.at/~agi/vahunz/.

If you think you found a bug, please check the known problems before.

Possible bugs also include problems with the manual, for example if something does not work as described. Furthermore this program is intended to be designed in a way so that it displays an error message for everything that can go wrong. If it does not, it's also a bug.

Always include the version number when reporting such things. You can obtain it either by invoking vahunz --version, or with the version command as usual.

As I wrote this program mainly to fit my own needs, I am not really interested in any comments and suggestions. If it turns out to be useful for you, use it, otherwise not. See also the chapter about possible future improvements.

So in case there is a reason to contact me, send an e-mail to Thomas Aglassinger <agi@giga.or.at>.
